# Overview

I picked Python because I had some exposure to it before and wanted to get comfortable with the basics before things get harder. This week I built a small greeting program — nothing fancy, but it helped me understand how functions with default parameters work and how to loop through a list without overcomplicating it.

The trickiest part was remembering that a `for` loop in Python doesn't need parentheses like in other languages. Kept second-guessing the syntax at first.
